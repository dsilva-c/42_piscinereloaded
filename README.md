This repository contains my work from the **Piscine Reloaded** project at **42 Porto**.

## üìå About The Project

The **Piscine Reloaded** serves as a vital bridge between the **intensive 26-day C Piscine** and the beginning of the **Common Core** curriculum.

It acts as a comprehensive refresher course, offering a "best-of" collection of exercises originally tackled during the Piscine. For students returning to 42 to start the main curriculum, this project provides the perfect opportunity to "dust off" their skills and ensure that the foundational concepts of the C programming language are fresh in their minds.

By revisiting these challenges, the project helps consolidate knowledge of:
* **Unix/Linux Shell** commands and file manipulation.
* **C Basics**: Variables, loops, functions, and the Norm.
* **Memory Management**: Pointers and allocation.
* **Data Structures**: Structs and headers.
